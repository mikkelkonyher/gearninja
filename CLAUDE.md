## Tech Stack

- **Frontend:** React (Vite), Tailwind CSS
- **Backend:** Supabase (Auth, DB, Storage)
- **Logic:** Supabase Edge Functions (Deno/TypeScript)

## Database Rules (CRITICAL)

- **READS:** You are permitted to use MCP tools to read the schema, tables, and rows at any time to gain context.
- **WRITES/UPDATES:** YOU MUST ALWAYS ASK FOR EXPLICIT PERMISSION before executing any SQL, creating tables, or modifying existing data.
- **SCHEMA:** Always use the MCP get_schema or equivalent tool before suggesting changes to ensure SQL accuracy.
